ASR 1002-HX Supported Network Interface Modules

Product ID Description
NIM-SSD The solid state drive (SSD) carrier card network interface module (NIM) enables SSD support on the platform NIM slot. It provides flash storage to the platform and supports
dual 2.5" (7mm max) SATA SSDs. The carrier card fits into the NIM slot in the router. The router supports only a single SSD Carrier Card NIM.
